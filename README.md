# Lightweight Fine-Tuning Critic Framework to Support Large Language Models in Specialized Medical Text Coding
This repo is for our paper "Lightweight Fine-Tuning Critic Framework to Support Large Language Models in Specialized Medical Text Coding".

## Dataset
Please access the DPICS-I dataset from [1] and DPICS-II dataset from [2] and [https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/C5Z3SC](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/C5Z3SC). Please download the raw Medical Abstracts Text Classification (MATC) dataset and the Sentiment Analysis for Mental Health (SAMH) dataset from [https://www.kaggle.com/datasets/chaitanyakck/medical-text](https://www.kaggle.com/datasets/chaitanyakck/medical-text) and [https://github.com/gennarocostantino/Sentiment-Analysis-for-Mental-Health](https://github.com/gennarocostantino/Sentiment-Analysis-for-Mental-Health).
